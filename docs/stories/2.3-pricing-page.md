## Story

**Como** usuario do ProspectAI,
**Quero** ver os planos disponiveis e assinar diretamente pela aplicacao,
**Para que** eu possa fazer upgrade do plano Free para Pro ou Scale sem sair do app.

---

## Acceptance Criteria

- [x] AC1: Pagina `/pricing` exibe 3 cards (Free, Pro, Scale) com features, precos e CTA
- [x] AC2: Card do plano atual do usuario esta destacado com badge "Plano Atual"
- [x] AC3: CTA "Assinar Pro" e "Assinar Scale" redirecionam para Stripe Checkout
- [x] AC4: CTA "Gerenciar Assinatura" (para Pro/Scale) redireciona para Stripe Customer Portal
- [x] AC5: Pricing page e acessivel sem auth (marketing) mas CTAs requerem login
- [x] AC6: Badge de plano visivel no header da aplicacao (ex: "Pro" pill ao lado do nome)
- [x] AC7: Componente de upsell reutilizavel que aparece quando usuario atinge limite (403 do paywall)
- [x] AC8: Early adopter banner "Primeiros 100: 19 EUR/mes para sempre" visivel na pricing page
- [x] AC9: Pagina responsiva e consistente com design system existente

---

## Scope

### In
