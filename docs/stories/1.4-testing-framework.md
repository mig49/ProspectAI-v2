## Story

**Quero** ter um framework de testes configurado com testes para os fluxos criticos,
**Para que** eu possa refatorar e adicionar features com confianca, sem regressoes silenciosas.

## Risk Assessment

| Risco | Mitigacao |
|-------|-----------|
| Next.js API routes requerem setup especial para testar (NextRequest mock) | Usar helper de mock padrao; documentar pattern nos testes |
| Mocking Gemini/Supabase pode ficar fragil | Mocks minimos, focar em input/output — nao mockar internals |
| 13 pontos e a maior story do epic | Dividir em tasks incrementais, rodar testes apos cada task |
