## Story

**Como** owner do ProspectAI,
**Quero** que as API routes verifiquem o plano do usuario e rastreiem o uso mensal,
**Para que** usuarios free tenham limites enforced e Pro/Scale tenham acesso completo as features.

---

## Acceptance Criteria

- [x] AC1: Tabela `usage` criada no Supabase com contadores mensais (searches, reports, exports)
- [x] AC2: Helper `lib/subscription.ts` exporta `getUserPlan(userId)` e `checkUsageLimit(userId, feature)` e `incrementUsage(userId, feature)`
- [x] AC3: `getUserPlan` retorna o plano ativo do usuario (default: 'free') com cache em memoria por request
- [x] AC4: POST `/api/prospect` incrementa `searches_count` e bloqueia com 403 se limite excedido
- [x] AC5: POST `/api/prospect/report` incrementa `reports_count` e bloqueia com 403 se limite excedido
- [x] AC6: Response de 403 inclui `{ error, plan, used, limit, upgradeUrl }` para o frontend mostrar upsell
- [x] AC7: Limites por plano configurados em `lib/constants.ts` (single source of truth)
- [x] AC8: Testes unitarios cobrem cada combinacao plano x feature x limite

## Technical Notes

### Limites por Plano

```typescript
// lib/constants.ts
export const PLAN_LIMITS = {
  free:  { searches: 3,   reports: 1,     exports: 0,  leads_per_search: 20 },
  pro:   { searches: 30,  reports: 9999,  exports: 30, leads_per_search: 50 },
  scale: { searches: 100, reports: 9999,  exports: 100, leads_per_search: 50 },
} as const;
```

### Tabela Usage

```sql
create table usage (
  id uuid primary key default gen_random_uuid(),
  user_id uuid references auth.users not null,
  month text not null, -- '2026-04'
  searches_count int default 0,
  reports_count int default 0,
  exports_count int default 0,
  created_at timestamptz default now(),
  unique (user_id, month)
);

alter table usage enable row level security;
create policy "Users read own usage" on usage for select using (auth.uid() = user_id);
create policy "Users update own usage" on usage for update using (auth.uid() = user_id);
create policy "Users insert own usage" on usage for insert with check (auth.uid() = user_id);
```

### Paywall Response Format

```json
{
  "error": "Limite de buscas excedido para o plano Free.",
  "plan": "free",
  "used": 3,
  "limit": 3,
  "upgradeUrl": "/pricing"
}
```
